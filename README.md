
# 👟 SkyStep

O **SkyStep** é uma aplicação web desenvolvida em PHP que simula uma loja virtual de tênis, permitindo visualizar produtos e organizar informações de forma dinâmica.

---

## 📌 Sobre o projeto

O SkyStep foi criado com o objetivo de praticar conceitos fundamentais de desenvolvimento web, especialmente:

* Manipulação de dados com PHP
* Estruturação de páginas dinâmicas
* Organização de código backend

O projeto representa um e-commerce simplificado, focado na lógica e funcionamento interno da aplicação.

---

## 🧠 Funcionalidades

* 👟 Listagem de produtos (tênis)
* 🔎 Filtro por categorias
* 📄 Exibição dinâmica de informações
* 🛒 Simulação de adição de produtos
* 📦 Organização de dados com arrays

---

## 🛠️ Tecnologias utilizadas

* **PHP** → Lógica backend
* **HTML5** → Estrutura
* **CSS3** → Estilização

---

## ⚙️ Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/skystep.git
```

2. Acesse a pasta do projeto:

```bash
cd skystep
```

3. Execute o projeto:

* Use XAMPP, WAMP ou Laragon
* Acesse no navegador:

```
http://localhost/skystep
```
